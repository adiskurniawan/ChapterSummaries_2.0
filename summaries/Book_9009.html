JS runtime error during init Symptom: Controls missing. Table wrappers not created. Interactive features broken.
How to confirm: Console shows uncaught exception (TypeError, ReferenceError). Disabling JS shows static layout.
Fix (minimal patch): Wrap init in DOMContentLoaded and try/catch. See code below.
Sticky header hidden / wrong offset Symptom: Thead appears behind top header when scrolling. Sticky not visible.
How to confirm: Computed `top` smaller than main header height.
Fix (minimal patch): Make `thead th { top: var(--main-header-height); }` and set `--main-header-height` to actual header height.
Sticky fails due to overflow/transform ancestor Symptom: Thead not sticking or sticking inside scroll area (not viewport).
How to confirm: Ancestor has `overflow:auto/hidden` or `transform` in computed styles.
Fix (minimal patch): Remove/relocate overflow or move sticky header out of that scroll container.
Alternative: use fixed header or clone header.
Virtualization or DOM replace Symptom: Rows missing or layout jumps.
How to confirm: Script replaces table DOM; CSS selectors no longer apply.
Fix (minimal patch): Ensure virtualizer keeps `thead` in DOM and preserves classes.
Invalid HTML (unclosed tags) Symptom: Layout breaks unpredictably.
How to confirm: Validator complains.
Fix (minimal patch): Fix markup. Close tags and keep table structure valid.
Clipboard / large content exceptions Symptom: "Copy All" crashes script mid-init.
How to confirm: Console error during copy or memory-related exceptions.
Fix (minimal patch): Add size guard. Use download stream instead of clipboard for very large exports.
