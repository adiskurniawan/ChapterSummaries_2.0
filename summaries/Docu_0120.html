Table 4
Component
Responsibility
convert.py Builds canonical caption objects (type, text, source file/line, table/figure index).
Passes them downstream as structured data.
sanitize.py Normalizes caption text.
Removes unsafe HTML.
Guarantees predictable quoting for JSON embedding.
core_fragments.py Defines CaptionFragment.
Generates deterministic IDs (e.g., table-17, fig-03).
Maintains mapping between Markdown source → fragment ID.
core_table.py Ensures every table gets a stable table_id.
Attaches the ID to the fragment model.
Zero caption logic here; only identity.
core_renderer.py When load_group renders a file or section, it bundles all caption fragments relevant to that HTML block.
Produces a JSON payload ready for injection.
html_renderer.py Emits the real HTML (tables, headings, markup).
Emits <script data-caption-payload type="application/json">…</script> adjacent to the rendered content.
No DOM manipulation, only payload emission.
script.core.js Reads all <script data-caption-payload> blocks.
Parses JSON.
Calls injectCaptions(payload) once per block.
script.table.js Locates the target DOM node using the stable ID.
Inserts caption elements at the correct final DOM position.
Handles positioning, alignment, and responsive layout.
