Code was split but not unified, creating duplicate and diverging implementations. Unify _getTableKey logic: prefer dataset.tvTableId - dataset.tvId - id - index. Use one keyed object for sortStates and originalRowOrders.

Caption element mismatch between CSS and scripts. Read caption from .table-caption, fallback to <h3>. Keep consistent for export and copy.
Multiple sort/UI handlers. Consolidate updateHeaderSortUI and sortTableByColumn into one canonical implementation.
Missing origHtml snapshots after mutation. Call ensurePerCellOrigHtml() after init and DOM updates.
