Salaries YJKN 2026 MAR Final Step-by-Step (40%)
Introduction
This procedure explains how to prepare the March 2026 salary workbook (Salaries YJKN 2026 MAR Final.xlsm), update sheet names and period labels, transfer the required data from the source workbook, apply the necessary filters and formatting, and copy the final results into the March summary sheets for completion and saving. This process already covers approximately 40% of all steps required to complete Salaries YJKN 2026 MAR Final.xlsm, and it should take about 7 minutes to finish.

Step 1
1. Make a copy of the May workbook and rename it
Copy Salaries YJKN 2026 MAY Please adjust.xlsm and rename the copy to Salaries YJKN 2026 MAR Final.xlsm.
2. Open both workbooks
Keep both files open.
3. Switch to the source workbook
Activate Salaries YJKN 2026 MAY Please adjust.xlsm.
4. Copy columns A to L from Salaries YJKN 2026 MAY Please adjust.xlsm
Select columns A to L and copy them.
5. Return to the target workbook
Switch back to the workbook being updated.
6. Paste the copied data starting at A1
Select cell A1 and paste the copied content into the active sheet.
7. Freeze the top rows
Select cell A5, scroll slightly upward, and freeze panes so the top rows stay visible.
8. Rename the summary sheet
Change Summary_MAY to Summary_MAR.
9. Rename the first monthly sheet
Change MAY_1 to MAR_1.
10. Update the period label
Go to cell J3 and type PERIODE MARET 2026.
11. Copy the same period label to the second monthly sheet
Copy cell J3, paste it into MAY_2, then rename that sheet to MAR_2.
12. Duplicate the second March sheet
Copy MAR_2, place the copy after the third sheet, and rename the copied sheet to MAR_3.
13. Save the workbook
Save the file to keep all changes.

Step 2
1. Open the destination workbook
Open Salaries YJKN 2026 MAR Final.xlsm. This is the workbook that receives the updated data.

2. Open the source workbook
Open Salaries YJKN 2026.xlsm. This workbook provides the data to copy.

3. Go to the MAR sheet
Select the MAR worksheet in Salaries YJKN 2026.xlsm.

4. Unprotect the MAR sheet
Remove sheet protection from MAR so changes can be made.

5. Set the outline view on the MAR sheet
Show no row levels and show 2 column levels to control the visible grouped data.

6. Filter the MAR sheet
Apply an in-place advanced filter on J5:BI245 using the criteria range Salaries YJKN 2026 MAR Final.xlsm Summary_MAR!B4:B189. This keeps only the rows that match the summary criteria.

7. Go to the MAR_1 sheet in Salaries YJKN 2026 MAR Final.xlsm
Select the MAR_1 worksheet. This sheet receives copied values and formatting.

8. Copy values from the source MAR sheet into MAR_1
Copy the values from J5:V245 in the source workbook’s MAR sheet into J5:V245 in MAR_1.

9. Copy formats for the same range
Copy J5:V245 from the source MAR sheet and paste only the formatting into MAR_1 starting at J5.

10. Repeat the same process for the next range
Copy the values from AA5:AE245 in the source MAR sheet into AA5:AE245 in MAR_1, then paste only the formats for that range.

11. Repeat the same process for the last range
Copy the values from AN5:AT245 in the source MAR sheet into AN5:AT245 in MAR_1, then paste only the formats for that range.

12. Copy the range W6:Z6
Copy the top row of that block so it can be filled down to the last row.

13. Fill W6:Z down to the last row
Paste the copied W6:Z6 range down through the full data area ending at the last used row in column W.

14. Copy the range AF6:AJ6
Take the first row of the AF:AJ block for duplication.

15. Fill AF6:AJ down to the last row
Paste the copied AF6:AJ6 range down through the matching rows in the sheet.

16. Copy the range AU6:AY6
Take the first row of the AU:AY block for duplication.

17. Fill AU6:AY down to the last row
Paste the copied AU6:AY6 range down through the matching rows in the sheet.

18. Copy to MAR_2 and MAR_3
Select all cells in the active sheet and copy the data.
Switch to MAR_2, select all cells, and paste the copied data.
Switch to MAR_3, select all cells, and paste the same copied data.
