Table 2
Component
Enhancement
Description
Copy Modal Modes
 Add tri-mode logic: Rendered, Raw, Hybrid. - Rendered: outputs visual text as seen on screen, with <br> converted to real newlines.
- Raw: preserves source syntax including <br>, Markdown pipes, and original whitespace.
- Hybrid: merges visual formatting but retains explicit <br> markers for clarity.
Mode Persistence
 Store user’s last selected copy mode in localStorage (tvCopyMode). Guarantees that user workflow remains consistent across sessions.
Per-Table Context
 Each per-table button triggers copyTable(btn, mode) where mode defaults to stored preference. Buttons work independently yet respect shared configuration.
Visual Confirmation
 Implement non-intrusive flash highlight around table (outline: 2px solid var(--accent) for 400 ms). Provides immediate copy feedback without modal reliance.
Source Fidelity
 Always prefer window.__tv_last_source as canonical text when available. DOM reconstruction used only if the table was generated dynamically and source not recorded.
