2 – LATAR BELAKANG — Why the reform and expected outcomes
Problem statement: Old PPh 21 rules used multiple overlapping calculation methods causing inconsistent outcomes and high administrative cost.Objectives:1) Simplify withholding for employers and payers2) Improve voluntary compliance by reducing errors3) Enable better automated validation in withholding systems4) Provide consistent outputs across payroll softwareDesign choices: Use deterministic TER tables instead of ad hoc aggregation. Apply monthly aggregation; daily prorating for partial months or one-offs. Compute taxable base → apply TER → subtract PTKP → produce withholding.Expected benefits: Reduced audit disputes, fewer correction submissions, lower operational cost, faster taxpayer service. KPIs: fewer employer queries, fewer correction SPT Masa, higher match rate with annual SPT.Operational implication: Payroll vendors release patches; employers test full-month, partial-month, FX, bonus, severance, and pension scenarios.

3 – OVERVIEW: PERUBAHAN DAN PENYESUAIAN PENGHITUNGAN — Scope, TER mechanics, categories
Scope changes: Include natura/kenikmatan, certain pension distributions for active employees, consolidated monthly reporting for fixed earnings.TER mechanics:• TER Monthly Category A/B/C for wage bands• TER Daily for prorated periods• TER One-off for bonuses/terminationCalculation flow:1) Determine gross taxable components2) Map recipient to TER category3) Apply TER (monthly/daily/one-off factor)4) Subtract monthly PTKP portion5) Apply rounding rule6) WithholdCategory assignment rules: Employment status, payment type, frequency. Examples provided in source.Examples (source):1) Monthly salary IDR 15,000,000 → TER Category A → apply table % → withholding2) Mid-year bonus → annualize → One-off TER → prorate to month3) Natura → monetary equivalent → include in grossEdge rules: FX payments use bank reference rate on payout date; nonresident employers follow Pasal 26.

4 – DASAR PENGENAAN DAN PEMOTONGAN — Tax base, PTKP, inclusion/exclusion rules
Tax base definition: Wages, allowances, natura/benefits in kind, bonuses, commissions, employer pension contributions (taxable portion). Periodic wages → monthly; accrued/unpaid items → follow entitlement vs payment rules with retroactive adjustments if necessary.Developer notes: Separate gross components, tag income codes, ensure PTKP mapping; use formula sheets and mind maps.

Taxable components: Base salary, fixed/variable allowances, natura, employer pension contributions (if taxable).Step algorithm:1) Aggregate monthly gross components2) Full month → use Monthly TER3) Apply monthly PTKP portion4) Separate one-offs → One-off TER / daily prorate5) Apply rounding (nearest Rp1 or Rp100)6) Issue bukti potong, record journal entriesPTKP example: Married, 1 dependent → annual PTKP X → monthly PTKP = X/12One-off payments / bonuses: Bonus IDR 50,000,000 → annualize → One-off TER → compute withholdingSeverance / termination: Identify component → check exemption or table → compute withholding (flowchart in source)Mid-year joiner/leaver: Daily TER prorate, e.g., join day 10/30 → apply factor 21/30FX & non-resident: Convert FX using official rate; foreign employers → Pasal 26 / treaty rulesImplementation checklist: Unit tests for standard month, partial month, bonus, severance, natura valuation, FX conversion.

7 – PENGHITUNGAN UNTUK PEGAWAI TIDAK TETAP — Rates, prorata, and reporting
Definition: Daily labor, contract workers, temporary staff, honoraria recipients who are not permanent employees.Withholding base: Per payment; multiple payments same month → aggregate per rules.Daily TER process:1) Calculate gross for payment period2) Convert to daily equivalent if needed3) Apply daily TER4) Subtract prorated PTKP if eligible5) Round and remitExample: Daily worker 10 days at IDR X/day → compute monthly equivalent → apply non-permanent TER category.Honorarium / short contracts: Aggregate multiple honoraria in month for correct withholding; source has worked example.Reporting & bukti potong: Issue per payment or period; lampiran provides format and required fields.Admin tips: Flag non-permanent payees, store contract period dates, implement aggregation window logic.

10 – PEMOTONGAN PPh PASAL 26 — Nonresident withholding, treaties, and computation
Definition & scope: Withholding on Indonesian-sourced payments to nonresidents (individuals/entities) for services, royalties, interest, specified payments.Tariffs & treaty application: Domestic Pasal 26 rates unless DTT provides lower rate. Required evidence: Certificate of Tax Residency, contracts, invoices.Calculation formula:1) Determine gross Indonesian-sourced payment2) Determine gross vs net basis per income type3) Apply statutory Pasal 26 or treaty rate4) Retain documentation if using treaty rateExample: Royalty gross IDR X → 10% domestic or 5% treaty → withhold accordingly.Procedural requirements: Issue bukti potong Pasal 26, withhold/remit using designated code, file SPT Masa Pasal 26, keep treaty docs for audit.Edge cases: Permanent establishment risk; if PE exists → tax treatment may change, withholding may convert to reporting obligations.
